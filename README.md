# Credit Risk Modeling & Explainability Platform

## Business Problem
Financial institutions require accurate and explainable credit risk
models to comply with regulations and manage default risk.

## Solution
Built a PD modeling framework using logistic regression and XGBoost,
enhanced with SHAP-based explainability for transparent decision-making.

## Tech Stack
Python, Scikit-learn, XGBoost, SHAP

## Key Outcomes
- Improved risk segmentation
- Transparent credit decisions
- Regulatory-friendly explanations

## Future Enhancements
- Model monitoring & drift detection
- Integration with credit bureau feeds
- Scorecard deployment
